I developed an interactive WebGIS platform to identify and visualize flood-prone areas based on rainfall and soil absorption data. The system was fully built using HTML, Tailwind CSS, JavaScript, PHP, MySQL, and Leaflet.js, covering both frontend and backend development. On the frontend, I designed a responsive interface with interactive maps, data input forms, and an admin dashboard for seamless data management. On the backend, I implemented real-time database integration to enable direct spatial data visualization. The development process followed the SDLC (Waterfall) methodology, starting from requirements analysis, system design, and implementation to deployment.
